pyrogroupbuys.com was created as a free service to the pyro community to help streamline the group buy experience. I saw an opportunity to take a fairly tedious process and improve it with some programming magic.

If you have any questions, comments, or ideas for new features, feel free to reach out to me at markmichon7@gmail.com.

Technical Information

This app was entirely written in the Python programming language, using the Django web framework to put everything together. The HTML is rendered with Django Template Language, and it utilizes a couple customized Bootstrap themes via Bootswatch. The application calls out to a local PostgreSQL database which stores the application's state. As of typing this, the apps currently uses ~5,200 lines of code. Pyrogroupbuys is powered by Ubuntu Server 20.04 LTS along with Apache Web Server and mod_wsgi. Check out the Statistics page if you haven't already, it also contains some cool information on the site.

The full source code of this application is available on Github, if you wish to inspect the code, run your own private instance of this site, or contribute to the project (that would be cool).
